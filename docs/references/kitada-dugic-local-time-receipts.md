# Kitada / Dugić Local-Time Receipts

## Status

This is a sober Kitada / Dugić note for the Maxwell-completion /
closure-machine program.

It does not claim that Kitada, Jeknić-Dugić, Arsenijević, or Dugić completed
physics, proved `Z0_AsBinary`, or supplied the final substrate. Their useful
contribution is sharper:

```text
time is not a universal outside parameter for the universe-as-one;
time is local, relational, and attributable to systems or subsystems.
```

That makes their work a direct correction to the naïve Laplace's Demon picture.

---

## Historical anchor

Laplace's Demon imagines an intellect that knows the complete state of the
universe at one instant and can compute all past and future states from the laws
of classical mechanics.

That picture quietly assumes something enormous:

```text
a whole-universe state can be specified at a time,
and that time can be used as if it were available from outside the universe
```

The Kitada / Dugić local-time direction attacks that assumption. In Kitada's
local-time framework, time is defined for local quantum-mechanical systems, while
the total universe has no associated time. In the later Kitada / Jeknić-Dugić /
Arsenijević / Dugić minimalist treatment, time is treated as neither fundamental
nor universal at the quantum-mechanical level, but consistently attributable to
approximately closed systems and their subsystems.

Through this repo's language:

```text
Universe-as-one has no outside clock.
No outside clock means no universal comparator.
No universal comparator means Laplace's Demon cannot stand outside the whole and read it at time t.
Time appears where systems can be compared, correlated, or used as clocks for one another.
```

## Machine receipt

The machine receipt is:

```text
no comparison -> no operational time
subsystem split -> possible comparison
two systems in relation -> clock / record / change receipt
local time -> subsystem-level bookkeeping
```

That is the clean version of the intuition:

```text
the universe must sub-select into at least two relational roles
```

Not necessarily two substances. Not necessarily metaphysical dualism. But at
least two distinguishable roles:

```text
clock / system
observer / observed
record / recorded
inside / boundary
state / comparator
yin / yang
```

A universe with no possible internal comparison has no way to operationalize
"before," "after," "rate," "record," or "change" from the inside.

## The Laplace correction

Laplace gives a powerful classical ideal:

```text
complete state + law -> complete prediction
```

Kitada / Dugić force the missing question:

```text
complete state according to what clock?
```

For a subsystem, the question can make sense:

```text
subsystem state relative to local time / clock / comparator
```

For the universe-as-totality, the usual demon picture breaks:

```text
there is no external time parameter for the whole universe to be read against
there is no outside comparator to say what the universe is doing all at once
there is no external witness that can hold the whole state without becoming part of it
```

Therefore the useful correction is:

```text
Laplace's Demon is not merely too smart to build;
it assumes the forbidden outside of the universe.
```

In this project, that matters because the closure machine cannot be a single
finished library viewed from nowhere. It has to generate local comparison,
record, and time from inside the process.

Maxwell completion asks:

```text

induction, phase, propagation, impedance, and records to be possible?
```

Kitada / Dugić add:

```text
whatever is running cannot rely on a universal external time parameter;
time and records must arise through local subsystem relations
```

That means a candidate closure machine must include:

```text
subsystem selection
local comparison
```

This makes Mead and Zurek even more important:

```text
Kitada/Dugić -> no whole-universe outside clock; local time requires subsystem relation
```

Together, they pressure the same missing primitive:

```text
running relational closure that generates time, record, and electromagnetic interface from inside
```

## Guardrail

This note should not overclaim.

```text
local time is not proof of Z0_AsBinary
subsystem comparison is not automatically electromagnetic substrate
yin/yang language is an interpretive compression, not a formal derivation
Laplace's Demon failing does not by itself prove any particular closure machine
```

The useful claim is narrower:

```text
if the universe has no outside time/comparator, then any real machine must
produce local comparison, local records, and local time internally
```

That is exactly the kind of requirement the Maxwell-completion program should
preserve.
