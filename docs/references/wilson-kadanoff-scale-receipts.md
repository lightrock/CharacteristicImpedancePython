# Wilson / Kadanoff Scale Receipts

## Status

This is a sober scale/effective-law note for the Maxwell-completion /
closure-machine program.

Wilson and Kadanoff do not supply the substrate and do not prove `Z0_AsBinary`.
Their useful contribution is:

```text
physics changes with scale, and effective laws can emerge by coarse-graining.
```

That makes them mandatory guardrails against confusing one descriptive level with
the whole machine.

---

## Machine receipt

The core receipt is:

```text
microscopic degrees of freedom
-> coarse-graining / blocking
-> effective variables
-> scale-dependent laws
-> fixed points / universality classes
```

Through this lens:

```text
not every detail survives scale change
some structures are irrelevant
some structures are relevant
some different micro-machines share the same macro behavior
```

---

## What Kadanoff adds

Kadanoff's block-spin intuition gives a concrete picture:

```text
group nearby degrees of freedom
replace them with effective variables
inspect what survives as scale changes
```

For this repo:

```text
bit-level structure -> grouped words -> chart-like layout -> catalog receipt
```

The question is:

```text
which structures survive grouping, scale change, and description change?
```

## What Wilson adds

Wilson makes renormalization a machine for scale flow.

```text
couplings flow with scale
fixed points organize behavior
universality explains why different microscopic systems can share macro laws
```

For Maxwell completion:

```text
field equations may be effective interfaces
constants may be scale-facing receipts
substrate may not look like the rendered law at observer scale
```

## Relation to Z0_AsBinary

`Z0_AsBinary` should ask scale questions explicitly:

```text
single bits
3-bit groups
auto-segmented words
orientation views

catalog-level token hits
whole-repo statistical controls
```

Wilson/Kadanoff ask:

```text
what survives coarse-graining?
what disappears as irrelevant detail?
what becomes universal across seeds or catalogs?
what is merely a scale artifact?
```

This is especially important because the project reads a 39-bit object as a
possible machine. A real receipt should have some stability under admissible
coarse-graining and scale translation.

---

## Guardrail

```text
effective law is not substrate
scale invariance is not proof
universality can hide many different micro-machines
coarse-graining can create artifacts
```

Wilson/Kadanoff therefore supply both permission and skepticism:

```text
permission: simple lower-level rules can yield stable higher-level law
skepticism: the same higher-level law may not identify the unique lower-level machine
```
