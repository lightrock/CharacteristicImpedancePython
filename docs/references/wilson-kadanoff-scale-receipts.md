# Wilson / Kadanoff Scale Receipts

## Status

This is a sober scale/effective-law note for the Maxwell-completion /
closure-machine program.

Wilson and Kadanoff do not supply the substrate and do not prove `Z0_AsBinary`.
Their useful contribution is:

```text
physics changes with scale, and effective laws can emerge by coarse-graining.
```

The repair language for this repo is sharper:

```text
renormalization flow may be an observer-facing trace of relative rates
across an information horizon.
```

That makes them mandatory guardrails against confusing one descriptive level with
the whole machine.

## Machine receipt

The core receipt is:

```text
microscopic degrees of freedom
-> coarse-graining / blocking
-> effective variables
-> scale-dependent laws
-> fixed points / universality classes
```

Through this lens:

```text
not every detail survives scale change
some structures are irrelevant
some structures are relevant
some different micro-machines share the same macro behavior
```

The process-first translation is:

```text
fast / internal / inaccessible relation updates
-> information horizon for the slower observer or coarser description
-> unresolved closure work is integrated out
-> visible parameters become effective rates
-> couplings appear to run with probe scale
```

In short:

```text
Wilson may be reading relative rates, not finding separate machines.
```

## Information horizons of relative rates

An information horizon of relative rates appears whenever one process cannot
resolve another process's internal update structure.

That does not require mysticism. It only requires a mismatch:

```text
observer / apparatus / coarse-grained variable has one admissible rate
hidden relation-update has a faster, finer, or otherwise inaccessible rate
```

The unresolved side becomes an effective parameter, noise term, coupling,
renormalized mass, boundary condition, or ensemble correction.

Through this language:

```text
renormalization = disciplined bookkeeping across rate horizons
running coupling = visible receipt of hidden relative-rate compression
fixed point = stable rendering under repeated rate/scale translation
universality = many hidden rate grammars collapse to the same visible receipt
```

Guardrail:

```text
"relative rates" is not an exemption from standard RG.
Any proposed closure machine must still reproduce beta functions, fixed points,
universality classes, effective couplings, and known failure modes of effective theory.
```

## Guardrail

```text
effective law is not substrate
scale invariance is not proof
rate language is not proof
universality can hide many different micro-machines
coarse-graining can create artifacts
relative-rate horizons can hide the mechanism instead of identifying it
```

Wilson/Kadanoff therefore supply both permission and skepticism:

```text
permission: simple lower-level rules can yield stable higher-level law
permission: one machine can render different effective rates across observer horizons
skepticism: the same higher-level law may not identify the unique lower-level machine
skepticism: calling a coupling a rate receipt does not derive the beta function
```
