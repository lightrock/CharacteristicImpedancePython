## Why this is not consciousness talk

The word `observer` is dangerous because it can invite mysticism. That is not the
claim here.

The sober claim is:

```text
observer = stable process that performs physical equivalencing and leaves receipts
```

An atom can do this without awareness:

```text
many possible incoming field configurations
-> finite admissible transition structure
-> quantized response or no response
-> recordable emission / absorption / scattering behavior
```

The atom is therefore an observer-process because it is a stable closure system
that makes a physical distinction between admissible and inadmissible exchange.

## Relation to Einstein photoelectric transfer

Einstein's photoelectric effect is a boundary-transfer receipt:

```text

frequency above threshold -> admissible transfer event
intensity -> event count/rate, not maximum energy per event
```

That is exactly the atom/process-observer point in apparatus form:

```text

```

So the photoelectric effect is not only a quantum clue. It is an observer-process
clue: material structure filters possible electromagnetic relations into
discrete admissible events.
